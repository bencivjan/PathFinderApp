Instructions

This is a pathfinding simulator that uses the A* pathfinding algorithm to find the shortest path between the start (blue) and end (red) nodes.


Click and drag the start or end nodes to reposition

Click and drag on empty nodes to make them walls

Click and drag while holding the "shift" key to get rid of walls

Use the "Visualize algorithm" toggle to show the algorithm in action
